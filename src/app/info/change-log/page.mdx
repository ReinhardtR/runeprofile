<Title title="New Hiscores Leaderboards" date="2024-04-12" />

The plugin now submits HiScores data from the "1 Defence Pure" and "Level 3 Skiller" Leaderboards. These HiScores can be shown in the HiScores panel on your profile, by selecting the game mode in the dropdown. They will only be shown if your account is included in those leaderboards.

<Title title="Technical Overhaul" date="2024-04-08" />

## Skills

<div className="flex space-x-4">
  - Level number on skills at the minimum level are now red. Intended to make
  pures stand out more.
  <img src="/assets/change-log/2024-04-08/skills-level-red.png" width={200} />
</div>

- XP numbers now use commas as the delimiter, i.e. 1,000,000 instead of
  1.000.000. _Thanks for the feedback!_

## Collection Log

- The selected tab and page is now reflected in the URL, so it can be shared as a link.

<div className="flex space-x-4">
  - Items now show KC for all places where it can be obtained when hovering.
  <img src="/assets/change-log/2024-04-08/collection-log-obtained-at-kcs.png" width={400} />
</div>

- Items are now stored in a more efficient way, where the same item data is used on all pages where that item appears. This means you don't need to open all Wilderness Boss pages with the plugin when you obtain a Dragon Pickaxe.

- Now shows a fallback icon when no item icon can be found. *Please report any missing icons!*

- Pages where no items have been obtained are now marked with a red color.

- Pages that have not been submitted are marked with a grey color.

<Title title="Updated Wilderness Bosses" date="2023-01-26" />

The Wilderness Boss Rework changed the names of the Collection Log pages of the wilderness bosses (Callisto, Venenatis and Vet'ion). This required some manual work on the stored profile data.

The data from the old pages should now be moved to the new ones. This includes kill counts and the kill counts which the items were obtained at. The data should be moved automatically.

If you dont see the page on your profile, try opening the page in-game and then updating your profile. If there is any issues you can send feedback or contact me on Discord, found at the bottom of the site.
